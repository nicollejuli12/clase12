Ejercicio 4

4. Realizar una aplicación web que permita saber si un usuario pasa o pierde una materia.
La aplicación debe pedirle al usuario que ingrese el nombre de la persona, el nombre de la
materia, y debe pedirle 3 notas.
El algoritmo debe calcular el promedio de las 3 notas y determinar con ese promedio si
pasa o no la materia.
Si el promedio es menor a 3.0, debe indicar que perdió la materia, pero si es igual o
superior a 3.0 debe indicar que paso la materia.
Debe usar colores como el rojo o el verde para indicar que si se pasa o se pierde la materia.
Ejemplos:

Input: [Nombre: Pedro, Materia:Programación, Nota 1: 4, Nota 2: 3.5, Nota 3: 5] Output: Felicitaciones Pedro, su nota es 4.16, Pasaste la materia Programación.
Input: [Nombre: Pedro, Materia:Sistemas Op, Nota 1: 1, Nota 2: 3, Nota 3: 2.3] Output: Lo siento Pedro, su nota es 2.1, No pasaste la materia Sistemas Op.
Input: [Nombre: Juan, Materia:Sistemas Op, Nota 1: 2.5, Nota 2: 3, Nota: 3.5 ] Output: Felicitaciones Juan, su nota es 3, Pasaste la materia Programación.
